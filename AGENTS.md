# AI Agent Instructions

This repository is a Codex skill distribution repository.

When a user asks you to install this repository as a skill, install only this skill path:

```text
skills/ppt-master
```

Do not install the repository root as a skill.

Preferred install command:

```bash
python3 ~/.codex/skills/.system/skill-installer/scripts/install-skill-from-github.py --repo firstlightwcx/GOLDWIND-skills --path skills/ppt-master
```

Equivalent GitHub tree URL:

```text
https://github.com/firstlightwcx/GOLDWIND-skills/tree/main/skills/ppt-master
```

After installation, tell the user to restart Codex or refresh the skill list if the new skill does not appear immediately.
